### How to Report a Bug

#### Access the Enhanced Reporter
1. Go to `Help > Report an Issue` in the menu bar
2. The modern issue reporter opens with a professional tabbed interface
3. Select the **Bug Report** tab for bug-related issues
4. Use the **Feature Request** tab for new feature suggestions
5. Browse the **View Issues** tab to see existing reports

#### Comprehensive Bug Report Form
**Required Information:**
- **Title**: Clear, concise summary with automatic validation
- **Description**: Rich text area with markdown support for detailed explanations
- **Steps to Reproduce**: Structured, numbered steps with validation
- **Expected vs Actual Behavior**: Side-by-side comparison fields
- **Environment Details**: Automatically populated system information

**Enhanced Attachments:**
- **Smart Log Integration**: Intelligent log file selection and privacy filtering
- **System Screenshots**: Detailed instructions for capturing relevant screenshots
- **Configuration Export**: Automatic inclusion of relevant application settings
- **Diagnostic Data**: Optional performance metrics and diagnostic information
