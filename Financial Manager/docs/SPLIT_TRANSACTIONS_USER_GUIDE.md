# Split Transactions - Quick User Guide

## What is a Split Transaction?

A split transaction lets you divide one purchase across multiple spending categories. Perfect for shopping trips where you buy different types of items.

---

## When to Use Split Transactions

✅ **Grocery Store**: Food + Household items + Personal care  
✅ **Department Store**: Clothing + Electronics + Home goods  
✅ **Gas Station**: Gas + Snacks + Car wash  
✅ **Online Shopping**: Books + Toys + Gifts  
✅ **Business Trip**: Business expenses + Personal meals + Entertainment  

---

## How to Create a Split Transaction

### Step 1: Add Transaction
Click "Add Transaction" button

### Step 2: Enter Basic Info
- Amount: Total purchase amount (e.g., $95.00)
- Description: Store name
- Account: Your bank account
- Type: Income or Expense

### Step 3: Enable Split
☑ Check "Split this transaction across multiple categories"

### Step 4: Edit Splits
Split dialog opens automatically:

1. **Add Split**: Click "Add Split" button
2. **Choose Category**: Select from dropdown (e.g., "Food")
3. **Enter Amount**: Type the portion for this category (e.g., $50.00)
4. **Repeat**: Add more splits until total is reached

### Step 5: Validate & Save
- Check the summary at bottom
- ✓ Green = Valid (splits match total)
- ⚠ Orange = Need more splits
- ✗ Red = Over the total
- Click "OK" when valid

## Tips & Tricks

### Quick Split Strategy
1. Enter the largest category first
2. Add remaining categories
3. Last split auto-fills remaining amount

### Editing Splits
- Click "Edit Splits" button anytime before saving
- Delete splits with "Delete" button in each row
- Categories are searchable in dropdown

### Budget Tracking
- Each split counts toward its category budget
- Budget colors still appear in transactions table
- Over-budget categories highlight in red

## Common Questions

**Q: Can I edit splits after saving?**  
A: Currently no - plan to add in future update. Delete and re-create transaction for now.

**Q: Do splits work with recurring transactions?**  
A: Not yet - splits only work with regular transactions currently.

**Q: What if splits don't add up exactly?**  
A: Dialog won't let you save until splits equal total (within 1 cent).

**Q: Can I split income transactions?**  
A: Yes! Same process works for income splits.

**Q: Will this mess up my existing data?**  
A: No - existing transactions remain unchanged. Splits are optional.

## Best Practices

### ✅ DO:
- Use splits for multi-category purchases
- Keep split categories consistent with budgets
- Add notes in description if needed
- Review tooltips to verify splits

### ❌ DON'T:
- Don't use splits for single-category purchases

- Don't forget to check totals before saving

## Need Help?

If splits aren't working:
1. Verify transaction amount is entered
2. Check all categories are filled in
3. Ensure amounts sum to total
4. Try closing and reopening split dialog
5. Restart app if issues persist
