# Audio Dubbing Quality Improvements - Implementation Summary\n\n## Date: April 2026\n## Changes Made to: `subtitle_tool.py`\n\n---\n\n## Problem Statement\n\nUser reported three critical issues with audio dubbing:\n1. **Original audio was completely lost** - Output had only dubbed track, no original language\n2. **Dubbed audio was choppy and super-speed** - Audio quality was severely degraded\n3. **English not set as default audio stream** - Players wouldn't auto-play dubbed track\n\n---\n\n## Root Causes Identified\n\n### Issue 1: FFmpeg Muxing Command Malformed\n**Location:** `_mux_translated_audio()` function\n**Problem:**\n```bash\n# OLD (BROKEN):\nffmpeg -i video.mkv -i dub.wav \\\n  -map 0 -map 1:a:0 \\\n  -c copy                    # <- This copies ALL streams as-is\n  -c:a:1 aac                 # <- But then this tries to override the codec\n  output.mkv\n```\nThe `-c copy` copies all streams without modification, so the codec override doesn't work. More importantly, it doesn't preserve original audio properly.\n\n### Issue 2: Aggressive Audio Stretching + Merged Text Synthesis\n**Location:** Two problems combined\n\n**Problem A - `_fit_audio_to_target_duration()`:**\n- Used aggressive `atempo` filter with 2.0x steps\n- No quality preservation filters\n- Caused audible distortion and artifacts\n\n**Problem B - `_synthesize_edge_tts_audio()`:**\n- Was synthesizing ALL translated text in one pass\n- No respect for segment timing/duration\n- Output speed couldn't be properly controlled\n- Resulted in \"mushed together\" audio\n\n### Issue 3: Stream Disposition Not Set\n**Location:** `_mux_translated_audio()` function\n**Problem:** New audio track added but not marked as default, causing players to play original audio first\n\n---\n\n## Solutions Implemented\n\n### Solution 1: Proper FFmpeg Muxing (Lines 3830-3903)\n\n**NEW Approach:**\n```python\ncmd = [\n    ffmpeg,\n    \"-i\", input_video,\n    \"-i\", dubbed_audio,\n    \"-map\", \"0:v\",      # Explicit: all video streams\n    \"-map\", \"0:a\",      # Explicit: all audio streams  \n    \"-map\", \"0:s?\",     # Explicit: all subtitle streams\n    \"-map\", \"1:a:0\",    # Explicit: dubbed audio to add\n    \"-c:v\", \"copy\",     # Copy video unchanged\n    \"-c:a:0\", \"copy\",   # Copy first audio unchanged\n    \"-c:a:1\", \"copy\",   # Copy second audio unchanged (if exists)\n    \"-c:a:2\", \"aac\",    # NEW dubbed audio as AAC\n    \"-metadata:s:a:2\", \"language=eng\",\n    \"-disposition:a:2\", \"default\",  # Mark as default\n    \"-disposition:a:0\", \"0\",        # Clear default from originals\n    \"-c:s\", \"copy\",\n    output\n]\n```\n\n**Benefits:**\n- Explicit stream mapping ensures all content preserved\n- Original audio streams are copied unchanged\n- New audio properly codec'd and marked\n- Disposition flags control which track plays by default\n- Preserves all metadata and subtitles\n\n### Solution 2a: Gentler Audio Tempo Adjustment (Lines 3467-3495)\n\n**OLD:**\n```python\nfilters = []\nwhile remaining > 2.0:\n    filters.append(\"atempo=2.0\")  # Huge jumps!\n    remaining /= 2.0\nwhile remaining < 0.5:\n    filters.append(\"atempo=0.5\")  # Still very aggressive\n    remaining /= 0.5\n```\n\n**NEW:**\n```python\nfilters = []\nwhile remaining > 1.5:\n    filters.append(\"atempo=1.5\")  # Smaller, gentler steps\n    remaining /= 1.5\nwhile remaining < 0.67:\n    filters.append(\"atempo=0.67\") # Gentler\n    remaining /= 0.67\nif abs(remaining - 1.0) > 0.01:  # Only adjust if needed\n    filters.append(f\"atempo={remaining:.4f}\")\n```\n\n**Additional improvements:**\n- Threshold check: skip tempo adjustment if < 10% different from target\n- Added lowpass filter for high-ratio stretching (>1.5x)\n- Better preserves audio quality and naturalness\n\n### Solution 2b: Per-Segment Edge TTS Synthesis (Lines 3419-3518)\n\n**NEW Feature:**\nModified `_synthesize_edge_tts_audio()` to support per-segment synthesis:\n\n```python\ndef _synthesize_edge_tts_audio(\n    self,\n    text: str,\n    voice: str,\n    rate: str,\n    output_path: Path,\n    segments: Optional[List[Dict[str, object]]] = None,  # NEW\n    temp_dir: Optional[str] = None,                       # NEW\n) -> None:\n    if segments and temp_dir:\n        # NEW: Synthesize each subtitle segment separately\n        for idx, seg in enumerate(segments):\n            # 1. Extract text for this segment\n            # 2. Synthesize just this text\n            # 3. Fit to exact segment duration\n            # 4. Add silence gaps between segments\n            # 5. Concatenate all together\n    else:\n        # Fallback: old behavior for compatibility\n```\n\n**Benefits:**\n- Each segment synthesized separately ensures proper timing\n- Silence gaps preserve original timing between speakers\n- Natural pacing that matches subtitle timings\n- Matches proven XTTS approach\n- Output .wav file is properly concatenated\n\n**Implementation Flow:**\n1. Calculate total video duration\n2. For each non-empty subtitle segment:\n   - Add silence for any gaps since last segment\n   - Synthesize this segment's text with Edge TTS\n   - Fit synthesized audio to exact segment duration\n   - Append to concatenation list\n3. Add tail silence if needed\n4. Concatenate all segments with FFmpeg concat demuxer\n5. Output as 22050Hz mono PCM WAV\n\n### Solution 3: Stream Disposition Flags\n\n**NEW in `_mux_translated_audio()`:**\n```python\nif lang_tag in [\"eng\", \"en\"]:\n    # Mark dubbed audio as default\n    cmd.extend([\n        f\"-disposition:a:{source_audio_count}\",\n        \"default\",\n    ])\n    # Clear default from originals\n    for i in range(source_audio_count):\n        cmd.extend([\n            f\"-disposition:a:{i}\",\n            \"0\",\n        ])\n```\n\n**Result:** Media players automatically play English dub first\n\n---\n\n## Code Changes Summary\n\n| Function | Lines | Changes |\n|----------|-------|----------|\n| `_build_atempo_filter_chain()` | 3467 | Gentler tempo steps (1.5x instead of 2.0x), skip if <10% needed |\n| `_fit_audio_to_target_duration()` | 3481-3495 | Added threshold check, lowpass filter for aggressive stretching |\n| `_synthesize_edge_tts_audio()` | 3419-3518 | Added per-segment synthesis with silence insertion and concatenation |\n| `_mux_translated_audio()` | 3830-3903 | Complete rewrite with proper stream mapping, codec handling, disposition |\n| `translate_audio_with_voice_match()` | 4120-4127 | Updated Edge TTS call to pass segments and temp_dir for per-segment mode |\n\n---\n\n## Testing Recommendations\n\n### Test 1: Audio Preservation\n```powershell\nffprobe output_video.mkv\n# Verify:\n# -
