# Company Logo Setup Instructions

This directory should contain the company logo files for the About Dialog.

## Required Files:

1. **company_logo.png** - PNG format company logo
   - Recommended size: 80x80 pixels or 128x128 pixels
   - Should be a clean, professional company logo
   - Used in the "Company" tab of the About dialog

2. **company_logo.ico** - ICO format company logo  
   - Should contain multiple sizes: 16x16, 32x32, 48x48, 64x64
   - Used for potential future icon applications
