# Text to Speech Widget - User Guide

## Overview
The Text to Speech (TTS) Widget converts PDF text content into spoken audio. This powerful accessibility tool helps with document review, multitasking, learning disabilities support, and creating audio versions of written content.

---

## Quick Start (Basic Users)

### Step 1: Add PDF Files
1. Click **"Add PDF Files"**
2. Select one or more PDF documents
3. Files appear in the list with page counts

### Step 2: Choose Voice Settings
1. **Voice Selection**: Choose from available system voices
   - Windows: David, Zira, and other SAPI voices
   - System Default: Uses OS default voice
2. **Speech Rate**: Adjust speed (slow, normal, fast)
3. **Volume**: Set audio level (0-100%)

### Step 3: Select Content
1. **Entire Document**: Read the whole PDF
2. **Page Range**: Specify pages (e.g., 1-5, 10-15)
3. **Current Page**: Read single page only

### Step 4: Start Playback
1. Click **"Start Reading"**
2. Use playback controls to pause, resume, or stop
3. Navigate between pages during playback

## Troubleshooting

### Common Issues:

**"No voices available"**
- Check Windows speech settings
- Install additional language packs
- Restart application to refresh voice list
- Verify Windows Speech Platform is installed

**"Text not reading correctly"**
- PDF may be image-based (needs OCR)
- Text encoding issues
- Try different PDF viewer for comparison
- Check document properties

**"Audio playback problems"**
- Check system audio settings
- Verify audio drivers are working
- Try different voice selection
- Restart audio system

**"Performance is slow"**
- Large PDF files may take time to process
- Close other applications
- Adjust quality settings
- Use page-by-page mode for large documents

### Voice-Specific Issues:
- **Robotic Sound**: Try different voice or adjust quality settings
- **Mispronunciation**: Some technical terms may not be pronounced correctly
- **Speed Issues**: Adjust rate settings for comfortable listening
- **Volume Problems**: Check both application and system volume levels
