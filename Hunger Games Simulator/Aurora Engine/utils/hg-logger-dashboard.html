📋 Setup Instructions

  1. Install both TamperMonkey scripts:
    • HG_Logger_Collector.user.js - Runs on all websites
    • HG_Logger_Dashboard.user.js - Runs on this page
  2. Visit websites to capture console logs
  3. Return to this page to view aggregated logs
  4. Use filters and search to find specific logs
⚠️ Note: If you don't see the dashboard UI above, make sure:
  • TamperMonkey is installed and enabled
  • The dashboard script is installed and active
  • The script has permission to run on file:///* URLs
  • Try refreshing the page (Ctrl+R or Cmd+R)
