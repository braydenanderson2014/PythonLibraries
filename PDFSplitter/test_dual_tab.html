Test HTML Document for Dual Tab Editor

This is a test HTML document to demonstrate the dual tab functionality in the PDF Editor.

Features

Editor Tab: Edit the raw HTML source code with syntax highlighting and modification tracking.
Preview Tab: See the rendered HTML output in real-time.
Modification Tracking: Tabs show * when modified.
Tooltips: Hover over tabs to see full file paths.

Usage Instructions

  1. Open an HTML or MD file in the editor
  2. Two tabs will be created automatically:
    • filename.html (Editor) - For editing
    • filename.html (Preview) - For viewing
  3. Edit the file in the Editor tab
  4. The tab title will show * when modified
  5. Switch to Preview tab to see the rendered output
  6. Use Ctrl+S to save changes
Note: The preview tab updates automatically when you save the file.
