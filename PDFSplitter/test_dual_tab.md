# Test Markdown Document for Dual Tab Editor

This is a test Markdown document to demonstrate the dual tab functionality in the PDF Editor.

## Features

### Editor Tab
- Edit the raw Markdown source code
- Syntax highlighting and modification tracking
- Full editing capabilities

### Preview Tab
- See the rendered HTML output from Markdown
- Real-time preview of formatted content
- Read-only view

### Modification Tracking
- Tabs show `*` when modified
- Tracks changes accurately
- Updates tab display immediately

### Tooltips
- Hover over tabs to see full file paths
- Works for both editor and preview tabs
- Improved user experience

## Usage Instructions

1. **Open a Markdown file** in the editor
2. **Two tabs will be created automatically:**
   - `filename.md (Editor)` - For editing
   - `filename.md (Preview)` - For viewing
3. **Edit the file** in the Editor tab
4. **The tab title will show `*`** when modified
5. **Switch to Preview tab** to see the rendered output
6. **Use `Ctrl+S`** to save changes

> **Note:** The preview tab updates automatically when you save the file.
